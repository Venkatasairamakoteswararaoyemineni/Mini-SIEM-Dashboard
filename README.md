# Advanced Mini SIEM Dashboard

A professional cybersecurity project that simulates a Security Information and Event Management (SIEM) system for monitoring, analyzing, and visualizing security events in real time.

---

# Project Overview

This project is designed to demonstrate core SOC (Security Operations Center) concepts including:

- Log collection
- Threat monitoring
- Security event visualization
- Threat scoring
- Attack detection
- Security analytics

The dashboard processes thousands of security logs and displays them through a modern web interface.

---

# Features

## Security Monitoring
- Real-time log parsing
- Severity-based event classification
- Threat score calculation
- Security event tracking

## Log Management
- Processes 10,000+ logs
- SQLite database integration
- Structured log storage
- Fast querying system

## Visualization
- Interactive dashboard
- Pie chart analytics
- Severity counters
- Security event table

## Threat Detection
- SQL Injection detection
- Brute force monitoring
- Malware activity detection
- Ransomware alerts
- Privilege escalation detection
- Suspicious login monitoring
